class bpd.DataFrame(**kwargs)[source]¶

Bases: object

Custom DataFrame Class; Pandas DataFrames with methods removed.

Example

>>> df = DataFrame.from_records([[1,2,3],[4,5,6]], columns=['a', 'b', 'c'])
>>> df.shape
(2, 3)
>>> df.assign(d=[1,2]).shape
(2, 4)
>>> df.loc[1, 'b']
5
